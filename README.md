# 🔮 AI Tarot Reading Platform

An AI-powered tarot reading application that simulates real tarot reading experiences through interactive UI and AI-generated interpretations.

---

## 🚀 Overview

This project is a product-oriented application that integrates:

- Interactive card drawing experience  
- Tarot reading logic (spread, positions, reversals)  
- AI-powered interpretation generation  

It provides users with a step-by-step tarot reading process — from asking a question to receiving a personalized interpretation.

---

## ❓ Problem

Traditional tarot systems are either:

- Static → fixed meanings without personalization  
- Human-dependent → require a tarot reader  

There is a lack of scalable, interactive, and personalized digital tarot experiences.

---

## 💡 Solution

This project builds a structured tarot reading system that:

- Guides users through the full reading process  
- Simulates real tarot spreads (e.g., past–present–future)  
- Generates dynamic interpretations using AI  

---

## ✨ Features

- 🎴 **Interactive Card Drawing**
  - Random card selection  
  - Upright / reversed card positions  

- 🧭 **Structured Reading Flow**
  - Multi-step process: Question → Draw → Interpretation  

- 🤖 **AI-generated Interpretation**
  - Converts card combinations into structured prompts  
  - Produces personalized readings  

- 🧩 **Modular Architecture**
  - Separation between UI, logic, and AI components  

---

## 🏗️ System Architecture

### 🔹 Frontend
- Next.js (App Router)
- React
- TypeScript

### 🔹 Backend Logic
- Java (Maven)
- Handles:
  - Tarot card logic  
  - Spread structure  
  - Prompt generation  

### 🔹 AI Layer
- Prompt engineering for tarot interpretation  
- Converts structured inputs into natural language outputs  

---

## 🛠 Tech Stack

- Frontend: Next.js, React, TypeScript  
- Backend: Java, Maven, Gson, OkHttp  
- AI: Prompt Engineering  
- Auth / DB: Supabase, NextAuth
